## About Me

I'm a backend-first software engineer specializing in secure web applications, authentication systems, and AI-assisted automation. I build with Node.js, Express, PostgreSQL, and React — applying OWASP principles, threat modeling, and modern cryptography from day one, not as an afterthought.

My current focus: hardening backend systems against real-world attack vectors, building modular security architectures, and integrating automation into developer workflows.
